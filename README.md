# Citas Web - Costuras Sabina Saraic

Aplicación de citas web para servicios de costura con integración de WhatsApp y envío de fotos en Base64.

## Características

- ✅ Envío de mensajes personalizados por WhatsApp
- ✅ Envío de fotos en Base64 (sin servidor externo)
- ✅ Compatible con Cloudflare Pages
- ✅ QR actualizado automáticamente

## Tecnologías

- React + Vite
- Tailwind CSS
- Base64 para imágenes
- Cloudflare Pages

## Scripts

```bash
npm run dev      # Desarrollo
npm run build    # Producción (compatible Cloudflare)
npm run preview  # Vista previa
```

## Deploy

El deploy es automático al hacer push a master via Cloudflare Pages.

## Sistema Anti-Cache

La aplicación incluye un sistema automático anti-cache que fuerza la recarga de la página sin usar versiones almacenadas en caché.
